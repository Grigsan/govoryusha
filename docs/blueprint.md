# **App Name**: Говоруша

## Core Features:

- Интерактивные доски: Доски с картинками для выражения базовых потребностей и эмоций.
- Конструктор фраз: Визуальный конструктор для составления предложений из иконок.
- Преобразование текста в речь: Озвучивание составленных предложений с использованием преобразования текста в речь.
- Редактор контента: Раздел для добавления собственных изображений и аудиозаписей опекунами.

## Style Guidelines:

- Primary color: Light pastel blue (#A7D9ED) to promote calmness and reassurance.
- Background color: Very light blue (#F0F8FF) as a soft background, almost white.
- Accent color: Soft lavender (#D0B4DE) to draw attention to interactive elements.
- Font: 'PT Sans', a humanist sans-serif font, to ensure good readability in both headlines and body.
- Использовать четкие, интуитивно понятные иконки, легко различимые для детей с аутизмом.
- Интерфейс должен быть максимально простым и интуитивно понятным, с минимальным количеством элементов на экране, чтобы избежать перегрузки информацией.
- Использовать деликатные и плавные анимации для обратной связи с пользователем, избегать резких или навязчивых эффектов.
