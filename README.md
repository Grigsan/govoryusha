# Говоруша - Новое приложение

Веб-приложение AAC (Augmentative and Alternative Communication) для детей с аутизмом.

## Функции

- **Интерактивные доски**: Доски с картинками для выражения базовых потребностей и эмоций
- **Конструктор фраз**: Визуальный конструктор для составления предложений из иконок
- **Преобразование текста в речь**: Озвучивание карточек и составленных предложений

## Установка

```bash
npm install
```

## Запуск

### Режим разработки (порт 9005)
```bash
npm run dev
```

### Продакшн сборка
```bash
npm run build
npm start
```

Приложение будет доступно на `http://localhost:9005`

## Деплой на хост

1. Соберите приложение: `npm run build`
2. Запустите: `npm start`
3. Приложение будет работать на порту 9005
4. Настройте ваш веб-сервер (nginx/apache) для проксирования на порт 9005

## Структура проекта

- `src/app/` - Страницы приложения
- `src/components/` - React компоненты
- `src/hooks/` - React хуки
- `src/lib/` - Утилиты и данные
